The application was written in Typescript using the following frameworks and technologies:

  • Angular 10 frontend
  • Bootstrap 4 UI components
  • Nest.js backend
  • SQLite db + TypeORM database
  • Passport.js + JSON Web Token authentication

    (not implemented yet)

This is a simple TODO manager web app. After creating your own profile, you can log in with your password and username to manage your profile and todo items.

You can execute CRUD operations to add, modify, remove and display todo items.
